# ASM-PlusPlus

ASM-PlusPlus is a C++ to MIPS assembly converter developed as part of a university coursework project on compiler design. 
The application provides a simple graphical interface where users can input a subset of C++ code and view the generated assembly instructions.

The project demonstrates how high-level programming constructs such as variable assignments, arithmetic operations, conditional statements, 
and loops can be translated into low-level assembly code. 
It was built to explore the fundamental concepts behind compilers and the process of converting high-level languages into machine-level instructions.
